# 前端可视化界面 (Frontend Web)

这里存放用于人机交互的 HTML、CSS 和 JavaScript 代码。

## 前端开发基础注意事项
1. **极简主义，压缩体积**：单片机（ESP 芯片）的 Flash 存储空间极其有限（通常只有几 MB），请尽量避免引入庞大的外部框架（如完整的 Vue、React 或大型图表库），推荐使用原生 JS (Vanilla JS) 开发。
2. **避免外部依赖**：由于环境监控设备可能运行在没有外部互联网的局域网中，**不要使用外部 CDN 链接**来加载 CSS 或 JS 库。所有需要的资源必须尽量写在文件内部或下载到本地。
3. **接口地址配置**：在本地电脑开发时，AJAX/Fetch 请求的地址可能是局域网 IP（如 `http://192.168.1.100/api/status`）；但最终烧录到板子上时，应该使用相对路径（如 `/api/status`），请注意代码切换。
4. **Mock 数据先行**：在后端接口还没写好之前，请在 JS 里先写一段“假数据”来测试页面 UI 是否正常工作。
