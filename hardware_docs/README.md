# 硬件与物理接线文档 (Hardware Docs)

这里存放本项目所有的硬件接线图、物料清单以及物理联调说明。

## 硬件开发基础注意事项
1. **电源安全第一**：ESP32 芯片的逻辑电平是 **3.3V**，但在连接某些传感器或外设（如 5V 继电器）时，请务必确认供电电压是否匹配，切勿将 5V 电压直接接入芯片的 GPIO 数据引脚，以免烧毁主控。
2. **强弱电隔离**：在连接散热风扇（特别是如果使用 12V 甚至 220V 风扇时），必须确保继电器的光耦隔离电路正常工作，严禁带电接线。
3. **断电接线**：每次修改面包板上的杜邦线连接前，请**务必先拔掉 USB 供电线**。
4. **引脚冲突**：NodeMCU 的某些引脚（如 D0/D3/D4/D8）在启动时有特殊电平要求，尽量避免将关键传感器接在这些引脚上，推荐使用 D1, D2, D5, D6, D7 等安全引脚。
